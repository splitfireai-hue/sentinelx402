I built a threat intelligence API that AI agents can pay for — per request.

No API key. No subscription. No human in the loop.

Here's the idea:

AI agents need real-time security data. But every existing threat API requires:
→ account creation
→ API key management
→ monthly subscriptions
→ enterprise sales calls

That friction kills autonomous workflows.

So I built SentinelX402:

→ Domain risk scoring (detects phishing in <300ms)
→ IP reputation checking (C2 and botnet detection)
→ CVE analysis with exploit probability
→ Powered by live threat feeds (OpenPhish, Feodo Tracker, URLhaus)
→ India-focused coverage (UPI fraud, bank spoofing)

The payment layer uses x402 — the HTTP 402 protocol backed by Coinbase.
Agents pay $0.01-$0.05 per request in USDC on Base. No accounts needed.

First 1,000 requests are free.

One API call can prevent a phishing attack worth thousands.
